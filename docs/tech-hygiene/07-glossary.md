# :material-book-alphabet: Glossary

Plain-English definitions for the terms used in these manuals. Use search (top right)
to find any word quickly.

| Term | Plain-English meaning |
| --- | --- |
| App | A program that does one job, like T3RRA Survey or Maps |
| Browser | The app you use to open websites, usually Chrome or Edge |
| Cache | A saved copy of a website stored on your device so it works offline |
| COM port | A connection point Windows uses to talk to a device like the GPS or DualEM. Each one has a number, e.g. COM20. |
| CSV | A simple data file that can be opened in Excel. Survey data is exported as CSV. |
| Desktop | The main screen you see when Windows starts |
| Field (T3RRA) | The paddock or area you are surveying. Not the same as a physical field. |
| Folder | A container that holds files, like a folder in a filing cabinet |
| Force restart | Holding the power button to shut down a frozen device |
| GPS fix | Confirmation that the GPS receiver has locked on to satellites and knows its location |
| Hotspot | Using a phone's mobile data to give another device internet access |
| IP rating | A rating for how well a device is sealed against dust and water. The ET45 is IP65. |
| MIL-STD-810H | A military durability standard for drops, vibration, heat and dust. The ET45 and Getac meet it. |
| RTK | Real-Time Kinematic. A high-accuracy GPS correction method. "RTK required" in T3RRA means you need this before surveying. |
| Shapefile (.shp) | A map file format that stores the boundary of a paddock |
| SharePoint | Microsoft's file storage system. Survey files sync here automatically. |
| Sleep | Screen off but the device still on. Press the power button to wake it. |
| Sync | Automatically copying files between the device and SharePoint |
| Taskbar | The bar along the bottom of the Windows screen |
| `$PDLS1` | The data the DualEM sends when it is connected and reading correctly. It should update five times per second. |

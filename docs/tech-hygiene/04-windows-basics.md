# :material-microsoft-windows: Windows basics

Everything you need to know about Windows to do your job. Nothing more.

!!! info "AT A GLANCE"
    Files live in folders. Apps live on the taskbar or the desktop. If you are lost,
    press the Windows key to get back to the start.

## The desktop and taskbar

`[CONFIRM: screenshot of the Getac desktop with the taskbar labelled.]`

The **desktop** is the screen you see when Windows starts. Think of it as your
workbench. The apps and files you use often sit here.

The **taskbar** runs along the bottom of the screen. It shows:

- Apps that are pinned, so they are always there.
- Apps that are open right now.
- The clock and system icons on the right.

## Files and folders

A **file** is a single document or piece of data, like a survey CSV or a photo.

A **folder** is a container that holds files. Think of it like a folder in a filing
cabinet.

## The Windows key

If you are ever lost in Windows, press the **Windows key**. It is at the bottom-left of
the keyboard and looks like a flag. It takes you back to the Start menu.

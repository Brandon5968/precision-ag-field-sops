# Surveying (DualEM + T3RRA)

*Soil survey with the DualEM and T3RRA Survey — field SOP*

This manual is the field reference for running a **DualEM** soil survey with **T3RRA
Survey (v2.279)** on the Getac, end to end — from gathering the kit to handing off
clean, exported data that lands with the GIS team via SharePoint. Use it on the job:
every chapter opens with a quick **AT A GLANCE** box, then the detailed steps.
